Phase 0 Labs

This directory contains foundational ATT&CK + STIX exploration labs.

The purpose of Phase 0 is:

understanding ATT&CK structure
learning STIX modeling
building graph intuition
understanding ATT&CK relationships internally

This phase intentionally avoids:

AI integration
semantic reasoning
ontology complexity
SIEM integrations

Current labs include:

reading ATT&CK STIX data
extracting techniques
extracting groups
extracting relationships
constructing simple graphs
basic graph traversal
